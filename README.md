# Manikumar Portfolio

Personal portfolio website built to showcase my projects, technical skills, and contact information.

## Live Demo

https://manikumar-portfolio.vercel.app/

## Tech Stack

HTML5 • CSS3 • JavaScript (ES6+) • React.js • Tailwind CSS 

## Features

* Responsive portfolio website
* Projects showcase
* Skills section
* Contact form for direct communication

## Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/manikumar-portfolio.git
```

Navigate to the project directory

```bash
cd manikumar-portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
